# Clang-Format

In this folder run

```
npm install
```

Install xaver.clang-format in VSCode

add the following line to your settings

```
  "clang-format.executable": "${workspaceRoot}/utils/node_modules/.bin/clang-format"
```

\${workspaceRoot}/node_modules/.bin/clang-format
